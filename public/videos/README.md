# Tutoriel Vidéo - Comment Mesurer Votre Terrasse

## Placement du Fichier Vidéo

Ce répertoire contient les vidéos de tutoriel pour le configurateur Storal.

### Fichier Attendu
- **Nom** : `tutoriel-mesure.mp4`
- **Format** : MP4 (H.264 codec, AAC audio)
- **Résolution Recommandée** : 1920x1080 ou 1280x720
- **Durée Recommandée** : 2-5 minutes
- **Taille Recommandée** : < 50MB

### Instructions

1. Enregistrez votre vidéo de tutoriel en format MP4
2. Renommez le fichier en `tutoriel-mesure.mp4`
3. Placez-le dans ce répertoire (`/public/videos/`)
4. La vidéo sera automatiquement disponible dans le modal du configurateur

### Contenu du Tutoriel - Points Clés à Couvrir

- **M1** (Haut) : Largeur du mur supérieur en mètres
- **M2** (Gauche) : Hauteur du mur gauche en mètres
- **M3** (Bas) : Largeur du mur inférieur en mètres
- **M4** (Droite) : Hauteur du mur droit en mètres

### Intégration

Le tutoriel s'affichera dans un modal avec :
- Lecteur vidéo HTML5 avec contrôles natifs
- Légende explicative
- Points clés résumés sous la vidéo
- Bouton de fermeture

### Déclenchement du Modal

Le modal s'affiche automatiquement quand :
1. L'utilisateur clique sur le bouton "▶️ Comment mesurer ?" (haut-droit du visualiseur)
2. L'utilisateur exprime de la confusion ou pose des questions sur les dimensions
3. Les mots-clés détectés : "ne comprends pas", "cotes", "dimensions", "M1", "M2", "M3", "M4", etc.

### Animation du Bouton

Le bouton d'aide palpite (pulse animation) pour attirer l'attention quand :
- L'IA détecte une possible confusion dans le message de l'utilisateur

### Propriétés de la Vidéo

- Responsive : S'adapte à toutes les tailles d'écran
- Contrôles natifs : Lecture, pause, volume, plein écran
- Téléchargement désactivé : Pour protéger le contenu
- Mobile-friendly : Fonctionne parfaitement sur téléphones et tablettes
