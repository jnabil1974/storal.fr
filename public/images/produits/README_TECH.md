# Images Techniques à Ajouter

## Structure attendue

Pour chaque produit, ajoutez **2 images JPG** dans le dossier `tech/` :

```
public/images/produits/
├── kissimy_promo/
│   └── tech/
│       ├── encombrement.jpg   ← Image 1 : Dimensions coffre
│       └── inclinaison.jpg     ← Image 2 : Angle et projection
├── kitanguy/
│   └── tech/
│       ├── encombrement.jpg
│       └── inclinaison.jpg
...
```

## Noms de fichiers requis

- **`encombrement.jpg`** : Schéma montrant les dimensions du coffre et l'encombrement mural
- **`inclinaison.jpg`** : Schéma montrant l'angle d'inclinaison et la projection maximale

## Recommandations

- **Format** : JPG
- **Résolution** : 1200x675px (ratio 16:9) minimum
- **Poids** : < 300 Ko par image
- **Fond** : Clair de préférence (les images sont affichées sur fond blanc)
- **Qualité** : 85%
